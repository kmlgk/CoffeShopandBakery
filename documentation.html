Installation

No build step required. This is a static HTML template — everything runs in the browser.

  • Unzip the download to any folder.
  • Open index.html — it redirects to pages/index.html.
  • For a smoother dev experience, use a local server (e.g. VS Code Live Server).
Fonts and icons load from CDNs. For fully offline use, self-host them in assets/fonts/ and swap the <link> tags.

Colors & Fonts

All theming lives in CSS variables at the top of assets/css/style.css:

:root {
  --primary:   #9B6A43;  /* roasted mocha */
  --secondary: #6E8B6E;  /* café sage     */
  --accent:    #D6A419;  /* honey gold    */
  --cream:     #F7EFE4;
  --dark:      #2A1E16;
  --font-heading: 'Fraunces', Georgia, serif;
  --font-body:    'Nunito Sans', sans-serif;
}

Change a value once and it cascades everywhere. Dark-mode overrides live in dark-mode.css under [data-theme="dark"].

Components

Common building blocks (all in style.css):

  • .btn + .btn-primary / -outline / -secondary / -lg / -sm — buttons
  • .category-card · .feature-card · .menu-card · .blog-card · .pricing-card · .testimonial-card — cards
  • .pricelist — dotted-leader menu rows with .pl-badge dietary tags
  • .chip + [data-chip-group] — selectable pills (party size, time slots)
  • .faq-item — accordion (auto-wired in main.js)
  • .hours-list — opening-hours table
  • [data-reveal] — fade-up on scroll · [data-count] — animated counters

Forms & Integrations

Any <form data-validate> gets client-side validation and a success toast. Customize the toast message with data-success="…".

To connect a form to a real backend, add an action and method:

<form action="https://formspree.io/f/your-id" method="POST" data-validate>
  ...
</form>
  • Contact / reservations: Formspree or Netlify Forms.
  • Newsletter: Mailchimp / ConvertKit embed on .newsletter-form.
  • Map: swap the OpenStreetMap iframe in contact.html for a Google Maps embed.

Dark Mode & RTL

Both toggles live in the navbar and persist to localStorage (brew-theme and brew-dir). Dark mode also respects the visitor's OS preference on first visit.

To force a default, set attributes on the <html> tag: data-theme="dark" or dir="rtl".
